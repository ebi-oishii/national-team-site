# national-team-site

「サッカー代表監督シミュレーター」のGitHub Pages公開サイトです。

## 公開URL

GitHub Pagesを `main` ブランチから公開すると、以下のURLをApp Store Connectに設定できます。

- Support URL: `https://ebi-oishii.github.io/national-team-site/support/`
- Privacy Policy URL: `https://ebi-oishii.github.io/national-team-site/privacy/`
- Terms: `https://ebi-oishii.github.io/national-team-site/terms/`
- Commerce disclosure: `https://ebi-oishii.github.io/national-team-site/commerce/`
- Disclaimer: `https://ebi-oishii.github.io/national-team-site/disclaimer/`

## サポート窓口

- `ebi.apps.support@gmail.com`

## GitHub Pages設定

1. GitHubのリポジトリ画面で `Settings` を開く
2. `Pages` を開く
3. `Build and deployment` の `Source` を `Deploy from a branch` にする
4. `Branch` を `main` / `/ (root)` にして保存する
5. 数分後、`https://ebi-oishii.github.io/national-team-site/` で公開されることを確認する
